# 🔥 AI 热点监控工具 - 前端

## 📖 项目简介

本目录为 AI 热点监控工具的前端应用，基于 React + TypeScript + Vite 构建。

## 🛠️ 技术栈

- ⚛️ React 19
- 📘 TypeScript
- ⚡ Vite
- 🎨 TailwindCSS
- 🔌 Socket.io (实时通信)
- ✨ Aceternity UI (炫酷组件)

## 🚀 快速开始

```bash
# 📦 安装依赖
npm install

# ▶️ 启动开发服务器
npm run dev

# 🔨 构建生产版本
npm run build
```

## 📁 目录结构

```
client/
├── src/
│   ├── components/     # 🎨 UI 组件
│   │   └── ui/         # ✨ Aceternity UI 组件
│   ├── services/       # 🔗 API 和 WebSocket 服务
│   ├── utils/          # 🔧 工具函数
│   ├── App.tsx         # 📱 主应用
│   ├── main.tsx        # 🚪 入口文件
│   └── index.css       # 🎨 全局样式
├── public/             # 📂 静态资源
└── package.json
```

## ✨ 主要功能

- ⚙️ 关键词管理界面
- 🔥 热点列表展示
- 🔍 多维度筛选和排序
- 📬 实时 WebSocket 推送
- 🌐 响应式赛博朋克风格 UI
